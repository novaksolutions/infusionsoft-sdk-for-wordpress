== Description ==

Integrate with the Infusionsoft API using the free Novak Solutions SDK. This plugin is a dependency for other Infusionsoft plugins.

Easily enter your Infusionsoft app name and API key. The full SDK will be available to any plugins that need it, or your own custom code.
